# TOOLS.md - Notas locais

Habilidades definem  how  ferramentas funcionam. Este arquivo é para  seu  específicos — o material que é único para sua configuração.

# # O Que Vem Aqui

Coisas como:

- Nomes das câmaras e locais
- SSH hosts e pseudónimos
- Vozes preferidas para TTS
- Nomes das salas
- Apelidos de dispositivos
- Qualquer coisa específica do ambiente

# # Exemplos

```markdown
### Cameras

- living-room → Main area, 180° wide angle
- front-door → Entrance, motion-triggered

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- Preferred voice: "Nova" (warm, slightly British)
- Default speaker: Kitchen HomePod
```

# # Por que se separar?

Habilidades são compartilhadas. A tua configuração é tua. Mantê-los separados significa que você pode atualizar habilidades sem perder suas notas, e compartilhar habilidades sem vazar sua infraestrutura.
