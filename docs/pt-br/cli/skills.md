#`openclaw skills`

Inspecione habilidades (abundado + espaço de trabalho + sobreposições gerenciadas) e veja o que é elegível vs requisitos em falta.

Relacionados:

- Sistema de competências: [Habilidades] /tools/skills
- Configuração das competências: [Skills config] /tools/skills-config
- ClawHub instala: [ClawHub] /tools/clawhub

## Comandos

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```
