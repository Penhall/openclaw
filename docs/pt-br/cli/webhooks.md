#`openclaw webhooks`

Assistentes e integrações Webhook (Gmail Pub/Sub, webhook helpers).

Relacionados:

- Webhooks: /automation/webhook
- Gmail Pub/Sub: [Gmail Pub/Sub] /automation/gmail-pubsub

## Gmail

```bash
openclaw webhooks gmail setup --account you@example.com
openclaw webhooks gmail run
```

Ver [Documentação do Gmail Pub/Sub]/automation/gmail-pubsub para mais detalhes.
