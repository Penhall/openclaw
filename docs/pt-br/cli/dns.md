#`openclaw dns`

Ajudadores DNS para descoberta de áreas amplas (Tailscale + CoreDNS). Atualmente focado em macOS + Homebrew CoreDNS.

Relacionados:

- Descoberta do portal:
- Configuração da descoberta de área larga: [Configuração] /gateway/configuration

Configuração

```bash
openclaw dns setup
openclaw dns setup --apply
```
