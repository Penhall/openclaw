#`openclaw setup`

Inicializar`~/.openclaw/openclaw.json`e o espaço de trabalho do agente.

Relacionados:

- Iniciando: [Iniciando] /start/getting-started
- Wizard: [A bordo] /start/onboarding

## Exemplos

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

Para executar o assistente através da configuração:

```bash
openclaw setup --wizard
```
