#`openclaw pairing`

Aprovar ou inspecionar pedidos de pareamento de DM (para canais que suportam pareamento).

Relacionados:

- Fluxo de pareamento: /start/pairing

## Comandos

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```
